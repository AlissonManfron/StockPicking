# StockPicking

> ## Descrição
>
> - Aplicativo para controle de ações, fiis, stoks, reits, etc..
> 
<br/>

> ## Tecnologias
>
> - linguagem: Kotlin
> - padrão de arquitetura: MVVM
> - chamadas assíncronas: Coroutines
> - lib de persistência: Room
> - lib de gráfico: https://github.com/PhilJay/MPAndroidChart
>
